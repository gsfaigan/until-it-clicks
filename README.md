# Until It Clicks

An interactive web application for visualizing algorithms, data structures, and programming concepts with real-time audio feedback and step-by-step pseudocode.

## Features

- **44 Sorting Algorithms** with visual bar representations
- **7 Search Algorithms** with target highlighting and elimination tracking
- **7 Pathfinding Algorithms** on interactive grids with obstacle placement
- **9 Data Structure Visualizers** including linked lists, trees, heaps, and queues
- **8 C++ Essentials Modules** covering I/O, memory, pointers, and Big-O complexity
- **6 Bash & Unix Modules** for terminal commands, permissions, and file operations
- Real-time audio feedback corresponding to element values
- Step-by-step pseudocode display with line highlighting
- Adjustable size, speed, and algorithm-specific parameters

## Installation

```bash
git clone https://github.com/gsfaigan/sorting-visualizer.git
cd sorting-visualizer
npm install
npm start
```

## Data Structures

- **Linked Lists** - Singly and Doubly with insert, delete, reverse operations
- **Stack & Queue** - Push/pop and enqueue/dequeue with LIFO/FIFO visualization
- **Heap** - Min/Max heap with insert, extract, and heapify
- **Priority Queue** - Priority-based operations with heap backing
- **Binary Search Tree** - Insert, delete, search with tree visualization
- **BST Recursive Algorithms** - Inorder, preorder, postorder, height, count, sum, min, max, search, validate, mirror, diameter, path sum, LCA, level order
- **N-ary Tree** - BFS, DFS, height, count, sum operations

## C++ Essentials

- **cin / cout / cerr** - Standard I/O stream operations
- **File Streams** - ifstream, ofstream, fstream file operations
- **String Streams** - istringstream, ostringstream, stringstream parsing
- **Stack vs Heap Memory** - Memory allocation and lifecycle visualization
- **Pointers & References** - Address manipulation and indirection
- **Types, Strings & Vectors** - Fundamental data types and containers
- **Big-O Visualizer** - Interactive complexity comparison (O(1) to O(n!))
- **Tower of Hanoi** - Recursive algorithm visualization

- **Permissions** - chmod, chown, user/group/other permissions

- **I/O Redirection & Pipes** - stdin, stdout, stderr, pipes
